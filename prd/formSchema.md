## Allowed input type

| Type       | HTML input                |
| ---------- | ------------------------- |
| `text`     | `<input type="text">`     |
| `email`    | `<input type="email">`    |
| `number`   | `<input type="number">`   |
| `textarea` | `<textarea>`              |
| `select`   | `<select>`                |
| `radio`    | `<input type="radio">`    |
| `checkbox` | `<input type="checkbox">` |
| `date`     | `<input type="date">`     |
